# RL Drone Agent

This project implements a Q-learning based autonomous drone agent that learns to navigate through a static 8Ã—8 grid environment while avoiding obstacles and efficiently reaching its target destination.

## Usage

- **Clone the Repository**:
```bash
git clone https://github.com/kayc0des/drone_confirmation2D.git
```

- **Change to working directory**:
```bash
cd drone_confirmation2D
```

- **Create virtual environment**:
```bash
python3 -m venv venv
source venv/bin/activate
```

- **Install Requirements**:
```bash
pip install -r requirements.txt
```

- **Run Play File**:
```bash
python3 play.py
```
